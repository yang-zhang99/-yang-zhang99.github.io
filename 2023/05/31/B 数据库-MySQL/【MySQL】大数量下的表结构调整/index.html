【MySQL】大数量下的表结构调整

发表于 2023-05-31 分类于 数据库-MySQL 阅读次数： Valine：

参考博客： https://blog.csdn.net/procrastination/article/details/116056306

大数据量表调整表结构时，有时间会非常慢并且会失败，所以一种比较好的方法是通过类似表修改。

正式环境操作数据一定要当心。

1. 创建需要修改表的类似表

2. 修改类似表的表结构和索引

3. 将 原始表数据 插入到 类似表 中

4. 更改表名

5. 数据对比，确定未丢数据

6. 确认没问题，删除旧表
